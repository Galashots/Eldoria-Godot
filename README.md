# Eldoria-Godot

Eldoria-Godot is a separate Godot 4.x prototype/spike used to evaluate whether Godot should become a future Eldoria mainline. The existing Eldoria-V2 web game remains separate and is read-only reference for this project.

The current milestone adds a reusable dialogue box and a small Elder quest loop to the movement, collision, and collection slice.

## Run

Open `project.godot` in Godot 4.x standard and press F5.

Controls: WASD or arrow keys to move, E to talk, and E, Enter, or Space to close dialogue.
