# Claude Code Guidance

This repo is worked on by several AI tools (Claude, ChatGPT/Codex, Gemini/Antigravity), so
the actual guidance lives in one tool-neutral file: **read `AGENTS.md` first.** It links to
everything else required before starting work (`docs/design/NORTH_STAR.md`,
`docs/CURRENT_STATE.md`, `docs/GODOT_SPIKE_DECISIONS.md`, and the relevant
`docs/agent-workflow/` files).

This file exists only so Claude Code's own auto-loaded-context convention picks up that
pointer automatically. Do not duplicate `AGENTS.md` content here — if guidance needs to
change, change it there so every tool sees the same rules.
